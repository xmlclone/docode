venv
======

.. contents:: 目录

基础说明
==========

.. code:: 

    # 会在/path/to/new/virtual下创建environment虚拟环境
    python -m venv /path/to/new/virtual/environment

    # 比如(windows下)
    python -m venv c:\users\<username>\desktop\venv1

    # 激活
    c:\users\<username>\desktop\venv1\Scripts\activate.bat

    # 后续通过pip安装的包，就会在这个虚拟环境下
