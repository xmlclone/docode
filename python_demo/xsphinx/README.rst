Sphinx Demo
================

.. contents:: 目录

基础使用
----------

.. code:: console

    # 安装
    pip install sphinx

    # 快速生成一个基础的目录
    sphinx-quickstart

    # 编译
    sphinx-build -M html source build
    
    # 如果make工具，也可以使用下面的make命令
    make html
